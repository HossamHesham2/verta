<p align="center">
A modern **task management app** built with Flutter featuring Glassmorphism UI, smooth animations, and offline support.
</p>

---

## ✨ Features

* ➕ Add, edit, and delete tasks
* 🗂 Organize tasks by categories
* 🎯 Priority system
* 🌙 Dark & Light mode
* 💎 Glassmorphism UI
* 📱 Responsive design
* 💾 Offline storage (Hive)
* 🔍 Search functionality
* 🎬 Smooth animations
* ⚙️ BLoC state management 

---

## 🛠 Tech Stack

* Flutter
* Dart
* flutter_bloc
* Hive
* get_it / injectable
* flutter_animate
* glassmorphism
* google_fonts
* flutter_slidable
* flutter_screenutil
